# Smart Water Management for Sustainable Agriculture 

A Flutter and Firebase-based Smart Irrigation System integrated with IoT for real-time monitoring and remote control of agricultural irrigation systems. This project helps automate water management using sensor data such as soil moisture, temperature, humidity, and water flow.

## 🚀 Features

- 🌡️ Real-time monitoring of temperature and humidity  
- 💧 Soil moisture-based irrigation control  
- 🚰 Remote water valve control via mobile app  
- 🔄 Live data synchronization using Firebase Realtime Database  
- 📱 Cross-platform mobile application (Android/iOS)  
- 🔐 User authentication system  
- 🌍 Device based system integration using unique Device ID  

---

## 🛠️ Technologies Used

- Flutter (Mobile App Development)
- Firebase Authentication
- Firebase Realtime Database
- Firebase Cloud Storage
- IoT Hardware Integration
- Android Studio

---

## 📡 System Architecture

- Mobile App (Flutter) communicates with Firebase  
- IoT sensors update real-time data to Firebase  
- Firebase acts as central data hub  
- App reads data and sends control commands
